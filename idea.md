
Название идеи: AI помощник для Plasma UI (библиотеки компонентов, которую мы будем использовать на "фронте" для всех наших платформенных сервисов). https://github.com/salute-developers/plasma

Проблема и Бизнес-ценность: Разработчики тратят до 20% времени на поиск информации в документации дизайн-системы и подбор правильных токенов. Ошибки в использовании гайдлайнов ведут к UI-багам и нарушению консистентности. Продукт: Интеллектуальный помощник, который интегрируется в среду разработки (IDE) и автоматизирует написание UI-кода на базе Plasma. Бизнес-ценность: Сокращение Time-to-Market для фич с UI на 15-20%, снижение порога входа для новичков и автоматический контроль соответствия дизайн-системе.

Основной пользовательский сценарий: Разработчик спрашивает в IDE: "Создай карточку продукта с изображением, ценой и кнопкой 'Купить', используя актуальные токены отступов и цветов". Ассистент (через MCP и GigaChat) анализирует актуальную документацию Plasma, извлекает нужные компоненты и токены, и генерирует готовый React-компонент, который можно сразу использовать в проде.

Критерии готовности (MVP):
- Реализован MCP-сервер, который локально индексирует текущую версию Plasma (доки и токены).
- Работает интеграция с GigaChat для генерации ответов.
- Демонстрация: Агент корректно отвечает на вопрос "Как использовать компонент Button?" и выдает код с правильными импортами.
